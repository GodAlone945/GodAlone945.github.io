# vite

## 构建项目

```powershell
npm init @vitejs/app
// 让vite支持vue2项目
npm install vite-plugin-vue2 -D
```

## 创建vue2项目-vite.config.js

```jsx
import { createVuePlugin } from 'vite-plugin-vue2'
// 安装 ‘vite-plugin-require-transform’ 可以使用require
import requireTransform from 'vite-plugin-require-transform';
import path from 'path'

// https://vitejs.dev/config/
export default ({
  plugins: [
    createVuePlugin(),
    requireTransform({
      fileRegex: /.js$|.vue$/
    })
  ],
  resolve: {
    alias: [
      {
        find: '@',
        replacement: path.resolve(__dirname, 'src')
      }
    ],
    extensions: ['.js', '.vue']
  }
})
```
