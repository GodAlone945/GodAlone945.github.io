# 1.1 子组件内部回调自己

当某个组件在其内部至少一次再次渲染自己时，无法获取到父组件定义的方法 解决办法

<!-- parent.vue -->
<Parent @nodeClick="nodeClick"/>
<!-- child.vue -->
// 在组件内部再次调用组件时，无法使用父组件传来的方法
// 使用 $listeners 获取到父组件传来的方法
<Child @nodeClick="$listeners.nodeSet"/>
