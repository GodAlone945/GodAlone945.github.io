```html
// 同样的内容添加两份
<div class="img">
    <img src="../assets/image 15.png" alt="">
    <img src="../assets/image 15.png" alt="">
 </div>
```

```css
.img img:nth-of-type(2) {
  display: none;
}

@keyframes wordsLoop {
  0% {
    
  }
  100% {
    transform: translateX(-100%);
  }
}

@media (max-width: 1200px) {
  .img {
    display: flex;
  }

  .img img:nth-of-type(1) {
    animation: 10s wordsLoop linear infinite normal;
  }

  .img img:nth-of-type(2) {
    display: inline-block;
    animation: 10s wordsLoop linear infinite normal;
  }
}
```
